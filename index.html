VicuñaUploader

Free tool designed to upload files to Wikimedia Commons and other Wikimedia projects

Download (ver. 1.19) Wikimedia Commons page

Features

  • saves metadata in session XML file, it is possible to prepare files for upload off-line
  • checks filename conflicts before sending files to Commons
  • supports choice of categories like Upload Wizard
  • supports geotagging – one-click on a map window
  • supports chunked uploads for big files
  • support internationalization via translatewiki.net – 20+ languages available
