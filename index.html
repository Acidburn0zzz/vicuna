VicuñaUploader is a free tool written in Java designed to upload files to Wikimedia Commons and other Wikimedia projects.

Features

  • Save & load session using XML file,
  • Edit multiple files description at once,
  • Category hints,
  • Pre-upload file check,
  • Multilingual support
